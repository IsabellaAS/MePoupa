FAQ

Perguntas Frenquentes

Como é feita a instalação do sensor?

Instalamos o sensor que mede o fluxo de água e uma placa lógica que passa a informação para o aplicativo. Para isso é feito uma pequena serragem no cano para a alocação do sensor, e a placa lógica é plugada junto ao sensor.

Vai precisar fazer obra?

Apenas se o cano não for exposto. Mas nesse caso, apenas um pequeno espaço necessita de obra para que o cano seja alcançado. Tudo é realizado em um único dia.

O aplicativo me avisa sobre vazamentos?

Não possuimos uma funcionalidade específica para isso. Mas através do Tempo Real é possível saber se alguma torneira está aberta, gerando a possibilidade de saber se há vazamentos.

Aonde eu baixo o app automatizado?

Temos uma versão do aplicativo na Play Store aonde você pode começar a controlar seus gastos de forma manual. Para ter a solução com as outras funcionalidades automatizadas nos contate para realizar um orçamento.
